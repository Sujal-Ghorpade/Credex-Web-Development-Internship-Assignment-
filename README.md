# SoftSell Landing Page 🚀

A modern, responsive single-page landing site for **SoftSell**, built using React + Vite + TailwindCSS..

---

## 🧠 Features Implemented

- ✅ **Hero Section** with CTA button
- ✅ **How It Works** – 3-step process with visuals
- ✅ **Why Choose Us** – 4 benefit cards with icons
- ✅ **Customer Testimonials** – 2 dummy reviews
- ✅ **Contact Form** – Name, Email, Company, License Type, Message (with frontend validation)

---

## 💅 Design Decisions

- **Tech Stack**: React + Vite + TailwindCSS
- **Styling**: Clean layout with modern UI, dark mode support, and Framer Motion animations
- **Mobile-first**: Fully responsive across devices
- **Color Palette**: Blue-gray palette with accessible contrast
- **Typography**: Tailwind's default sans-serif system stack

---

## 🛠 Bonus Features

- ✅ Logo placeholder in header
- ✅ Favicon and page title
- ✅ SEO meta tags
- ✅ Light/Dark Mode support
- ✅ Framer Motion animation
